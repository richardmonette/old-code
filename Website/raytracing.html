Stochastic Path Tracing / Distribution Ray Tracing

For the CSI5146 Advance Computer Graphics course at Ottawa University, I implemented a stochastic path tracing renderer as my term project.

The renderer I implemented has the following features:

  • Moller-Trumbore Ray Triangle Intersection
    Fast, minimum storage ray/triangle intersection
  • Octree Space Partitioning

    • Using the octree data structure reduces the number of ray-triangle intersection tests required accelerating rendering

  • Barycentric Normal Interpolation
  • .obj Scene File Support
  • Lambertian Diffuse BRDF
  • Pure Specular (Mirror) BRDF
  • Reflection/Refraction BRDF
  • Explicit Emitter Sampling

    • Since the location of all emitters in the scene is known, it is possible to use an extra ray cast to a randomly sampled emitter in order to reduce the number of samples required to converge on the 'correct' solution. In order to ensure that the light contribution is not duplicated the direct emission is only included at the first sample.

  • UV Texture Mapping
