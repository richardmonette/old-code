Hello World!

I am Richard Monette, a computer graphics programmer/student currently working on my Master's of Science (Information System Science) at Carleton University under the supervisor of Dr. Anthony Whitehead.

Prior to starting my Master's I was employed as a game developer at Fuel Industries in Ottawa, Canada. I hold Bachelor's of Information Technology (Interactive Multimedia and Design specialization) with Distinction from Carleton University.

Areas of Interest

  • Photon Mapping

    Using the photon mapping algorithm to produce high quality, consistent and physically plausible images in practical render times more

  • Stochastic Path Tracing / Distribution Ray Tracing

    Using the standard, unbiased Monte Carlo integration techniques more

  • Real-time Graphics via Programmable Shaders

    Using HLSL/GLSL in the Direct3D and OpenGL pipelines to create shader effects for real-time 3D content such as games and visualizations. more

Projects, Productions and Work

  • Implementation of a few tone mapping operators
  • Software Rendering (CPU only transform, lighting, rasterization
