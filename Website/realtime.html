Realtime Graphics

In addition to my offline, production rendering technique projects I have worked on a number of real-time game oriented projects.

Some highlights include:

  • Glasshouse - An XNA based first person adventure game featuring custom HLSL shaders (deferred rendering, post process, etc.)
  • OpenGL .bsp Engine - A game engine tech demo which uses Quake III .bsp maps, nVidia PhysX and GLSL shaders.
